# Examples — 練習用リポジトリ

このフォルダには、ローカルで手を動かして学べる練習用リポジトリの雛形と自動セットアップスクリプトがあります。

目的:
- Git の初期化、ブランチ作成、コンフリクト発生と解消のハンズオンを素早く試せる

使い方:
1. ターミナルでこのフォルダに移動します:

```bash
cd examples
```

2. セットアップスクリプトを実行します:

```bash
bash Example_Local_Operation.sh
```

3. `demo-practice` ディレクトリが作成され、いくつかのコミットとブランチが用意されます。手順はスクリプトの出力に従ってください。

内容:
- `demo-practice/` — 自動で作られる練習リポジトリ
- `Example_Local_Operation.sh` — リポジトリを初期化して練習用コミットやブランチを作成するスクリプト
- `.gitignore` — 練習用に追加

注意:
- スクリプトはローカルのみの操作を行います（リモートへ push はしません）。
- 実行前に必要なコマンド (`git`, `sed`) が利用可能か確認してください（Linux 環境を想定）。
